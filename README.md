# NextArbiter

Vite + React + TypeScript starter for NextArbiter, styled with Tailwind CSS and shadcn components.

## Stack

- Vite
- React
- TypeScript
- Tailwind CSS
- shadcn components

## Scripts

```bash
npm install
npm run dev
```

Build for production:

```bash
npm run build
```

Run deterministic browser tests:

```bash
npm run test:e2e
```

If Playwright browsers are not installed yet:

```bash
npx playwright install chromium
```
